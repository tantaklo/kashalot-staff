## Записи

### 2026-05 — Netlify недоступен у российских пользователей

- **Симптом:** Сайт открывается на kashalot-staff.netlify.app только через VPN
- **Причина:** Netlify заблокирован провайдерами в РФ
- **Решение:** Использовать только GitHub Pages URL
- **Проверка:** Открыть tantaklo.github.io/kashalot-staff с российской SIM без VPN
- **Где связано:** `CLAUDE.md`, `decision-log.md`

### 2026-05 — README.md в корне — это файл темы, не проекта

- **Симптом:** При попытке обновить README.md можно случайно затереть документацию темы astro-theme-pure
- **Причина:** Проект склонирован из репозитория темы, её README лежит в корне
- **Решение:** Не трогать `README.md` и `README-zh-CN.md` в корне. Документация проекта — только в `docs/`
- **Проверка:** `docs/README.md` существует и заполнена
- **Где связано:** `docs/README.md`

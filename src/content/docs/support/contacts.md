## Бот технической поддержки

Основной канал для технических вопросов:

👉 **[t.me/franch_sup_bot](https://t.me/franch_sup_bot)**

Пиши туда, если:
- снегокат не едет и самодиагностика не помогла
- нужна замена запчасти
- что-то непонятно в работе техники

## Рабочий чат франчайзи

Для оперативных вопросов по смене — обращайся в рабочий Telegram-чат своего франчайзи.  
Контакт чата даёт франчайзи при выходе на работу.

## Что сообщить при обращении

Чтобы тебе помогли быстрее, укажи:
1. Номер или ID снегоката
2. Что именно происходит (или не происходит)
3. Что уже пробовал
4. Фото или видео проблемы — если есть

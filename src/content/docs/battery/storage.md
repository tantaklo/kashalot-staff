Актуально в конце сезона. Правильное хранение сохраняет батарею до следующей зимы — срок до 6–7 месяцев при соблюдении условий.

## Подготовка к хранению

1. Извлечь АКБ из снегоката
2. Зарядить до **50–60%** — оптимальный уровень
3. Очистить корпус от грязи, снега, соли сухой тканью
4. Осмотреть: вздутия, трещины, повреждения разъёмов → при находке сообщить в поддержку
5. Проверить баланс ячеек через **Xiaoxiang BMS**

## Условия хранения

- Температура: **+10...+20°C** (допустимо 0...+25°C)
- Сухое, проветриваемое помещение без прямого солнца
- Зазор между батареями не менее 5 см
- Не подключать к снегокату во время хранения
- Не хранить рядом с металлическими предметами (риск замыкания)

## Таблица напряжений

Измерять в покое, без нагрузки:

| Заряд | Напряжение | Рекомендация |
|---|---|---|
| 100% | 53,8 В | Только для работы, не для хранения |
| 70–80% | 51,0–52,2 В | Максимум для длительного хранения |
| **50–60%** | **49,8–50,6 В** | ✅ Оптимально |
| 40–45% | 48,9–49,4 В | Нижняя граница |
| < 40% | < 48,0 В | ⚠️ Опасно — срочно зарядить |

## Контроль во время хранения

- Проверять напряжение каждые **2–2,5 месяца**
- Если упало ниже 49,0 В → подзарядить до 50–52 В
- При хранении выше +20°C → проверять ежемесячно

## Запуск нового сезона

1. Перенести АКБ в тёплое помещение (+10°C и выше) — минимум на **6–8 часов**
2. Полностью зарядить штатным ЗУ
3. Установить на снегокат, проверить работу на малой мощности
4. Проконтролировать баланс ячеек через BMS после первой зарядки
